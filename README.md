## List of Experiments

### Experiment 1: ML Environment Setup
- Installation and configuration of Anaconda
- Setup of Jupyter Notebook, Spyder, VS Code, and PyCharm
- Installation and version checking of NumPy, SciPy, Pandas, Scikit-learn, and Matplotlib
- Understanding the use of these libraries in machine learning

### Experiment 2: Data Pre-Processing
- Downloading datasets from UCI, Data.org, or other public repositories
- Data cleaning and preprocessing
- Handling missing values
- Encoding categorical data
- Data normalization and scaling
- Basic data visualization

### Experiment 3: Artificial Neural Network (ANN)
- Implementation of ANN Classifier
- Implementation of ANN Regressor
- Display of training loss graph
- Performance evaluation using:
  - Confusion Matrix
  - Precision
  - Recall
  - F1-Score

### Experiment 4: Bayesian Classifier
- Development of Bayesian classification model
- Probability-based prediction
- Evaluation on dataset

### Experiment 5: Principal Component Analysis (PCA)
- Implementation of PCA using Breast Cancer dataset from scikit-learn
- Dimensionality reduction
- Feature variance analysis

### Experiment 6: Decision Tree
- Decision Tree Classification and Regression
- Model training and testing
- Query-based prediction

### Experiment 7: Support Vector Machine (SVM)
- Implementation of SVM classifier
- Kernel-based classification
- Model performance evaluation

### Experiment 8: K-Means Clustering
- Implementation of K-Means algorithm
- Clustering of multidimensional data
- Visualization of clusters

### Experiment 9: K-Nearest Neighbors (KNN)
- Implementation of KNN algorithm
- Performance evaluation
- Error rate comparison for different K values (1 to n)

### Experiment 10: Convolutional Neural Network (CNN)
- Implementation of CNN using TensorFlow and Keras
- MNIST handwritten digit dataset
- Model training and accuracy evaluation

### Experiment 11: Image Processing using OpenCV
- Basic image processing techniques
- Image reading and resizing
- Filtering and edge detection using OpenCV

### Experiment 12: Case Study
- Case study on Generative AI and Conversational AI
- Real-world applications and use cases
